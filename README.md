# 🧠 Memory Card Game (React)

A fun and interactive **Memory Card Matching Game** built using **React.js**.
The goal of the game is to match all the pairs of cards by remembering their positions.

---

## 🎯 How to Play

1. Click on any card to flip it.
2. Select another card to find its pair.
3. If both cards match, they stay open.
4. If not, they flip back.
5. Match all cards to win the game.
